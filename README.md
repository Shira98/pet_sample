# WineShop example: Order Management Using Blockchain

## Users/Contracts -

 - Customer
 - Shop
 
 ## Functionalities -
 
 ### Customer:
 
 - Can add item to cart.
 - Can checkout/pay.
 
 ### Shop:
 
 - Can add a wine bottle into the shop.
 - Can create new carts for customers.
 - Can provide a cutomer order's delivery date.
